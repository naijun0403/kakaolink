# kakaolink
KakaoTalkShare API for kakaobot

## 사용된 라이브러리
- [crypto-js](https://github.com/brix/crypto-js)
> 이후 패키지 관리자가 들어올때까지 직접 빌드해서 사용합니다.
>
> 빌드 방법은 [여기](docs/HOW_TO_BUILD.md)를 참고해주세요.

## v1
v1과 v2는 서로 호환되지 않습니다.

이 점 유의해주시고 사용해주세요.
